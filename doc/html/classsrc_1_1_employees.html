Detailed Description

Represents a collection of employee with a fixed maximum capacity.

This class is designed to store instances of the Employee class in a fixed-size array, defined by the constant sizeArrays. The array is statically initialized upon first access to the Employees class.

Example of use

Employees employees
Employees.AddEmployee(new Employee employee("Antonio Pereira","Eletricista", 6.3))
