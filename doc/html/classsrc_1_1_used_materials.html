Detailed Description

The UsedMaterials class represents the materials used in a project. It stores the details of the material (id,name, price) and the quantity used in the project, along with the date when the material was added.

The UsedMaterials class manages a collection of materials used in a project, storing each material with its quantity and the date it was used. The materials are stored in an array with a fixed size, and the total number of materials used is tracked.

Example of use:

UsedMaterials.AddMaterial(new Material("Cemento", 5.0), 10);
UsedMaterials.ShowList();
