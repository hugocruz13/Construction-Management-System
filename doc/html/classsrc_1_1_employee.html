Detailed Description

Class Employee allows to create an employee object with basic information such as their Id, name, role, and hourly role.

This class inherits from Person and provides methods to access and modify the employee attributes, and assign an Id automatically using a static counter.

Example of use

Employee emp = new Employee("Hugo Cruz", "Engineering", 12.3);
