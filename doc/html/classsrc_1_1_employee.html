Detailed Description

Purpose: Class Employee allows to create an employee object with basic information such as their Id, name, role, and hourly role.

This class allows to create an employee and provide methedos to access and modify the employee's attributes.

Example of use

Employee emp = new Employee();
emp.Id = 1;
emp.Name = "Hugo";
emp.Role= "Engineering";
emp.HourlyRate = "12.3";
