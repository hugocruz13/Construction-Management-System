Detailed Description

The Client class allows you to create a client object with basic information such as id, name and contact information.

This class inherits from Person and provides methods to access and modify the client attributes, and assign an Id automatically using a static counter.

Example of use

Client c = new Client("Maria Rodrigues", "9323423553")
