Detailed Description

The class Material represents a material used in processes, with attributes such as ID, name and unit price. The ID is automatically generated, and the unit price is validated to ensure it is positive.

The class provides properties to access and modify the attributes of a material, as well as a method to check the equality of two materials based on the name.

Example of use

Material mat = new Material("Cimento", 4.80)
