Detailed Description

Represents a collection of client with a fixed maximum capacity.

This class is designed to store instances of the Client class in a fixed-size array, defined by the constant sizeArrays. The array is statically initialized upon first access to the Clients class.

Example of use

Clients clients
Clients.AddClient(new Client client("Ricardo Alves","96......."))
