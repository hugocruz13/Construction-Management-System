Detailed Description

Represents a company with static attributes for managing clients, material inventory, employees, and projects.

The Company class provides a centralized, static structure for managing the core entities of a business. As all members are static, no instances are required.

Example of use

Company.ProjectCreate;
Company.EmployeeRegister;
