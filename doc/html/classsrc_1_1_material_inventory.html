Detailed Description

Represents a inventory with a fixed maximum capacity.

This class is designed to store instances of the MaterialQuantity class in a fixed-size array, defined by the constant sizeArrays. The array is statically initialized upon first access to the MaterialInventory class.

Example of use

MaterialInventory inventory
MaterialInventory.AddMaterial(new MaterialQuantity("Cimento", 4.80, 5))
